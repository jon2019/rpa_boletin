# Legacy documental

Acá se guardó material histórico que fue validado y clasificado como **no vigente** frente al estado actual del proyecto.

## Criterio

Un archivo termina acá cuando pasa alguna de estas cosas:

- describe un schema anterior
- omite tablas o columnas hoy existentes
- replica exactamente otro archivo y no agrega valor
- fue generado como entrega puntual (docx/xlsx/png/sql) y ya no debe usarse como fuente de verdad

## Regla práctica

- **Usar** `docs/operacion`, `docs/negocio` y `docs/datos` para documentación vigente.
- **No usar** `docs/legacy` para tomar decisiones de diseño sin volver a contrastar con el código.
